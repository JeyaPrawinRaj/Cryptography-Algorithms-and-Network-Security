# 🔐 Cryptography Algorithms and Network Security

This repository contains implementations of various **Cryptography Algorithms** and **Network Security lab exercises**, covering classical, modern, and public key cryptography techniques along with intrusion detection using Snort.

---

## 📘 Exercises Included

### 🔹 Exercise 1: Classical Cryptography & Number Theory

* Caesar Cipher
* Shift Cipher
* Playfair Cipher
* Vigenère Cipher
* Hill Cipher
* Primality Test
* Euclidean Algorithm (GCD)
* Primitive Root

---

### 🔹 Exercise 2: Modern Cryptography

* AES (Advanced Encryption Standard)
* DES (Data Encryption Standard)

---

### 🔹 Exercise 3: Public Key Cryptography

* Diffie-Hellman Key Exchange
* RSA Algorithm

---

### 🔹 Exercise 4: Hashing Algorithms

* MD5 Algorithm
* SHA-512 Algorithm

---

### 🔹 Exercise 5: Network Security using Snort

* Installation and configuration of Snort IDS
* Packet capturing from live network and PCAP files
* Writing custom rules
* Detection of:

  * Network scans (SYN, UDP, FIN, NULL, XMAS)
  * ICMP (Ping) traffic
  * HTTP requests and attacks
  * DNS queries and anomalies
* Attack simulation using Nmap

---

## 💻 Technologies Used

* Python
