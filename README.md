# 🔐 Cryptography Algorithms and Network Security

This repository contains implementations of various **Cryptography Algorithms and Network Security lab exercises**, covering classical, modern, and public key cryptography techniques.

---

## 📘 Exercises Included

### 🔹 Exercise 1: Classical Cryptography & Number Theory

* Caesar Cipher
* Shift Cipher
* Playfair Cipher
* Vigenère Cipher
* Hill Cipher
* Primality Test
* Euclidean Algorithm (GCD)
* Primitive Root

---

### 🔹 Exercise 2: Modern Cryptography

* AES (Advanced Encryption Standard)
* DES (Data Encryption Standard)

---

### 🔹 Exercise 3: Public Key Cryptography

* Diffie-Hellman Key Exchange
* RSA Algorithm

---

## 💻 Technologies Used

* Python
* HTML

* To understand and implement cryptographic algorithms
* To learn encryption and decryption techniques
* To study secure communication methods
